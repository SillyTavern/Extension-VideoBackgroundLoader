# Video Background Loader

Allows to perform video background uploads. Uses @ffmpeg/wasm to convert videos to animated webp format.

## Installation

Install using the following link in SillyTavern extension installer:

```txt
https://github.com/SillyTavern/Extension-VideoBackgroundLoader
```
